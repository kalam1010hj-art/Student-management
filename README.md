# Student Management System 🎓

A full-stack web application built with **Django** to manage student records efficiently. This project features a secure authentication system and dynamic data handling.

## 🚀 Features
- **User Authentication:** Secure Login/Logout system for staff and students.
- **CRUD Operations:** Create, Read, Update, and Delete student profiles.
- **Form Validation:** Used Django ModelForms to ensure data integrity.
- **Responsive UI:** Clean interface built with HTML/CSS.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite (Development)

## 📋 How to Run Locally
1. Clone the repo: `git clone https://github.com/kalam1010hj-art/Student-Management.git`
2. Create a virtual environment: `python -m venv venv`
3. Activate venv: `venv\Scripts\activate` (Windows)
4. Install Django: `pip install django`
5. Run migrations: `python manage.py migrate`
6. Start server: `python manage.py runserver
